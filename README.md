# Features ğŸ”¨

This is a 2D platformer game:
  - The character can move using the arrow keys.
  - The character can jump using the space bar.

## Ennemies
There are two types of enemies:
  - Bees / flies: if you jump on them, they are defeated.
  - Blobs: they are invincible, so you must avoid them.

## Obstacles
  - Spikes are deadly: touching them results in death.
  - Falling into the void also causes the character to die.

## Coins
  - You can collect coins throughout the levels.
  - Mystery boxes can be opened by jumping on them, granting additional coins.

## Levels
  - The game features two levels.
  - To access the second level, you must complete the first one.To access the second level, you must complete the first one.

## Menu 
  - Press P to start the game from the menu.
  - Press Q to quit the game.
